WON #1 PLACE in the 2025 VFS Project of Ignite. Designed a search & rescue eVTOL to drop a 5 lb package to a stranded hiker at 10,000 ft under 20–40 kt winds, 500 ft cloud cover, and fog cutting ½ visibility.
Constraint Analysis — Iterative Sizing Method

Iterated through 2,362,692 sizing configurations varying MTOW, S (wing planform area), A (total lift propeller area), and lift rotor speed fraction

  1. 1)airspeed > stall × FoS — airfoil selection (S1223) + lift rotor speed fraction
  2. 2)Total rotor ⌀ + propeller spacing + fuselage width < wingspan → lift rotor count
  3. 3)Energy fraction = stored ÷ required (power × time in cruise/hover/ascent/descent + drag)
  4. 4–6)CL verified; AOA via linear interpolation; W/S and D/L minimized
Applied Parameter Weights + Array Sort → MTOW: 224.81 lb · S: 2.7 m² · A: 40 m² · Speed Fraction: 90%

  • Aerodynamic calcs for transport cruise (18 m/s), search cruise (8 m/s), and hover — propeller radius, pitch, tip speed, motor torque, shaft RPM. T/W = 1.5
  • Researched spiral-in, spiral-out, and lawnmower search algorithms; coded 2,000 average and max search time simulations
