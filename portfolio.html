AERODYNAMICSPYTHONCONSTRAINT ANALYSISCAD (FUSION 360)FORMAL PRESENTATIONTECHNICAL REPORTSLICING (3D PRINT)

WON #1 PLACE in the 2025 VFS Project of Ignite. Designed a search & rescue eVTOL to drop a 5 lb package to a stranded hiker at 10,000 ft under 20–40 kt winds, 500 ft cloud cover, and fog cutting ½ visibility.

Constraint Analysis Iterative Sizing Method: Iterated through 2,362,692 sizing configurations, varying MTOW, S (wing planform area), A (total lift propeller area), and lift rotor speed fraction

  • 1) airspeed > stall × FoS: optimized by airfoil selection (S1223) and lift rotor speed fraction
  • 2) Total rotor ⌀ + propeller spacing + fuselage width < wingspan → number of lift rotors
  • 3) Energy fraction = energy stored ÷ required (power × time in cruise/hover/ascent/descent + drag)
  • 4) Verified CL within range; 5) AOA via linear interpolation; 6) minimized W/S and disk loading D/L

Applied Parameter Weights + Array Sort → MTOW: 224.81 lb · S: 2.7 m² · A: 40 m² · Speed Fraction: 90%

Additional Contributions
  • Aerodynamic calcs for transport cruise (18 m/s), search cruise (8 m/s), and hover: propeller radius, pitch, tip speed, motor torque, shaft RPM. T/W = 1.5
  • Researched spiral-in, spiral-out, and lawnmower algorithms; coded 2,000 average and max. search time simulations
  • Received aircraft frontal area from CAD; input sensor weight/power data into code
  • Split CAD into pinned components; used Bambu Studio for spliced 3D printing
